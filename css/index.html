css相关

  • nth-of-type与nth-child的区别：
  • a:nth-of-type(1){color:red;}:选取第1个a标签应用样式，非a标签不在计算范围内
  • p:nth-of-type(2){color:red;}:选取第2个p标签应用样式，非p标签不在计算范围内
  • a:nth-child(1){color:red;}:选取符合（第1个标签 && a标签）的标签，如果存在则应用样式，否则不应用
  • p:nth-child(2){color:red;}:选取符合（第2个标签 && p标签）的标签，如果存在则应用样式，否则不应用
    input,button,select,textarea 标签常用
  • 取消点击时的边框：outline:none;
    a 标签常用
  • 取消a标签点击时的颜色：
    -webkit-tap-highlight-color: transparent;
    -webkit-touch-callout: none;
    -webkit-user-select: none;
  • 取消display:inline-black;导致的内容间的空格
    letter-spacing:-*px; font-size:0;
